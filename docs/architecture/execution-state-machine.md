# Execution State Machine

## States

```
DRAFT
INTENT_CONFIRMED
DISCOVERING
RESEARCHING
RANKED
PROPOSAL_READY
PROPOSAL_EXPIRED
APPROVED
REJECTED
EXECUTION_QUEUED
PREFLIGHT
APPROVE_PENDING
APPROVE_CONFIRMED
BURN_PENDING
BURN_CONFIRMED
ATTESTATION_PENDING
ATTESTATION_READY
MINT_PENDING
MINT_CONFIRMED
COMPLETED
RECOVERABLE
MANUAL_REVIEW
FAILED
CANCELLED_BEFORE_SIDE_EFFECT
```

## Valid Transitions

```
DRAFT → INTENT_CONFIRMED
DRAFT → CANCELLED_BEFORE_SIDE_EFFECT

INTENT_CONFIRMED → DISCOVERING
INTENT_CONFIRMED → CANCELLED_BEFORE_SIDE_EFFECT

DISCOVERING → RESEARCHING
DISCOVERING → RANKED (if no intelligence needed)

RESEARCHING → RANKED

RANKED → PROPOSAL_READY
RANKED → PROPOSAL_EXPIRED

PROPOSAL_READY → APPROVED
PROPOSAL_READY → REJECTED
PROPOSAL_READY → PROPOSAL_EXPIRED

APPROVED → EXECUTION_QUEUED

EXECUTION_QUEUED → PREFLIGHT

PREFLIGHT → APPROVE_PENDING (for CCTP approve)
PREFLIGHT → BURN_PENDING (if approve not needed)

APPROVE_PENDING → APPROVE_CONFIRMED
APPROVE_PENDING → FAILED

APPROVE_CONFIRMED → BURN_PENDING

BURN_PENDING → BURN_CONFIRMED
BURN_PENDING → RECOVERABLE
BURN_PENDING → FAILED

BURN_CONFIRMED → ATTESTATION_PENDING

ATTESTATION_PENDING → ATTESTATION_READY
ATTESTATION_PENDING → RECOVERABLE

ATTESTATION_READY → MINT_PENDING

MINT_PENDING → MINT_CONFIRMED
MINT_PENDING → RECOVERABLE
MINT_PENDING → FAILED

MINT_CONFIRMED → COMPLETED

RECOVERABLE → EXECUTION_QUEUED (recovery attempt)
RECOVERABLE → MANUAL_REVIEW

FAILED → MANUAL_REVIEW (if recoverable components exist)
```

## Rules

- Valid transitions only — no skip-ahead
- Transactional state update — atomic with audit write
- Audit every financial transition
- One active execution per approved proposal
- Same execution lineage for recovery
- COMPLETED is immutable
